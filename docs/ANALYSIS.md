# Анализ аналогов и целевой аудитории

## Анализ конкурентов

### GitHub Copilot Chat
- **Сильные стороны**: Огромная обучающая выборка, интеграция с GitHub, поддержка множества языков
- **Слабые стороны**: Только облачные модели, нет кастомизации под проект, $10-19/мес, данные отправляются на серверы Microsoft
- **Уникальное**: Глубокая интеграция с GitHub workflow

### Cursor
- **Сильные стороны**: Отличный UX, контекст всего проекта, быстрый
- **Слабые стороны**: Форк VS Code (отдельное приложение), облачные модели, $20/мес для Pro
- **Уникальное**: Codebase-aware chat, Composer для мультифайловых правок

### Cody (Sourcegraph)
- **Сильные стороны**: Мощный RAG по кодовой базе, поддержка больших репозиториев
- **Слабые стороны**: Зависит от Sourcegraph, сложная настройка для enterprise
- **Уникальное**: Масштабируемый RAG через Sourcegraph graph

### Continue.dev
- **Сильные стороны**: Open-source, поддержка локальных моделей, расширяемость
- **Слабые стороны**: Менее полированный UX, нет мультиагентной системы
- **Уникальное**: Полностью открытый, поддержка Ollama

## Целевая аудитория AI Code Partner

### Основная ЦА
1. **Команды с требованиями к безопасности данных**
   - Банки, финтех, госсектор, медицина
   - Код не должен покидать периметр организации
   - **Наше преимущество**: Полностью локальный инференс

2. **Малые и средние команды разработки**
   - Не могут позволить себе $20/чел/мес за Cursor/Copilot для всей команды
   - **Наше преимущество**: Бесплатно, модель крутится на одном сервере для всей команды

3. **Команды с устоявшимся coding style**
   - Строгие конвенции именования, архитектурные паттерны
   - AI-инструменты генерируют код "не в стиле" команды
   - **Наше преимущество**: Fine-tuning на конкретном проекте

### Уникальные преимущества

1. **Приватность**: Весь код остаётся на машине разработчика
2. **Кастомизация через fine-tuning**: Модель перенимает стиль команды
3. **Мультиагентная система**: Специализированные агенты для разных задач вместо одного универсального
4. **Бесплатность**: Нет подписок и лимитов токенов
5. **Поддержка слабого железа**: 4-bit квантование позволяет работать на CPU или слабых GPU
